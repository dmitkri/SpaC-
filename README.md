# Управление СПА Салоном

Проект по С++. Управление СПА салоном. 5 семестр.

## Версия для GitHub Pages

Этот проект адаптирован для работы на GitHub Pages без сервера. Используется mock API с хранением данных в localStorage браузера.

### Как опубликовать на GitHub Pages

1. Перейдите в настройки репозитория на GitHub: https://github.com/dmitkri/SpaC-/settings/pages
2. В разделе "Build and deployment":
   - **Source**: выберите "Deploy from a branch"
   - **Branch**: выберите `main` и папку `/ (root)`
3. Нажмите **Save**
4. Через несколько минут ваш сайт будет доступен по адресу: 
   `https://dmitkri.github.io/SpaC-/`

**Важно:** Используйте обычный статический хостинг (Deploy from a branch), а не GitHub Actions!

### Структура файлов

- **HTML файлы** - страницы приложения:
  - `home.html` - главная страница
  - `index.html` - панель управления (для администраторов)
  - `login.html` - вход и регистрация
  - `client.html` - онлайн запись для клиентов
  - `booking.html` - форма записи
  - `client-dashboard.html` - личный кабинет клиента

- **JavaScript файлы**:
  - `mock-api.js` - эмуляция API сервера (использует localStorage)
  - `script.js` - логика панели управления
  - `client.js` - логика клиентской части
  - `booking.js` - логика формы записи

- **Другие файлы**:
  - `style.css` - стили приложения
  - `.nojekyll` - отключает обработку Jekyll на GitHub Pages
  - `.gitignore` - исключает C++ файлы из репозитория

### Особенности версии для GitHub Pages

- Все данные хранятся в localStorage браузера
- Не требуется сервер - приложение полностью клиентское
- Данные сохраняются локально в браузере пользователя
- Все функции API эмулируются через mock-api.js

### Оригинальная версия (с C++ сервером)

Если нужно запустить версию с C++ сервером:
- Скомпилируйте сервер: `make`
- Запустите: `./spa_server`
- Откройте в браузере: `http://localhost:8081`
